Installation
============

From GitHub source:

.. code-block:: bash

    git clone https://github.com/lyx12311/OpenSoftwareTut.git
    cd OpenSoftwareTut
    python setup.py install


Dependencies
============

The dependencies of *OpenSoftwareTut* are
`NumPy <http://www.numpy.org/>`_,

These can be installed using pip:

.. code-block:: bash

    pip install numpy
